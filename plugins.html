Plugins let you add new gameplay features to your server and modify existing features.

Install Plugins

There are many plugins available to use, and installing them is easy. You can find most plugins available for Cuberite on our plugin repository, or on the forums. The User's Manual contains instructions on how to install a plugin.

Develop Plugins

Although there is a large number of plugins to choose from, you may not always find exactly what you want. In this case you can request a plugin in our forum, or create a plugin yourself.
