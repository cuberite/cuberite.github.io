Cuberite is licensed under the Apache License V2, and we welcome anybody to contribute to the project. If you want to join as a permanent member, we can add you to the team. Check out the GETTING-STARTED.md file for more details, and visit our GitHub repository to start developing.

If you're not a developer, you may also donate to help the project. Donations will first cover the web hosting fees needed for keeping the build and download servers running; the rest will be distributed among team members.
