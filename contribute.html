If you feel like contributing to the Cuberite project, there are various ways you can do so.

Code

We welcome any interested developers to contribute to the project. Check out the Getting Started guide for more information about development.

Getting Started

Report Bugs

Bug reports help us keep track of what should be fixed on or added to the Cuberite server. If you've found a bug or a missing feature, feel free to report it in the bug tracker.

Bug Tracker

Donate

If development isn't your thing, you may also donate to help the project. Donations will be used to cover web hosting expenses and to reward developers.

Donate
