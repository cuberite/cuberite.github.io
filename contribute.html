Cuberite is licensed under the Apache License V2, and we welcome anybody to fork and submit a Pull Request back with their changes. If you want to join as a permanent member, we can add you to the team. Check out the CONTRIBUTING.md file on Github for more details, and visit our Github repository to start developing. If you want to create plugins for Cuberite instead, you will find all the information you need in the API Documentation.

If you're not a developer, you may also donate to help the project. Donations will first cover the web hosting fees needed for keeping the build and download servers running; the rest will be distributed among team members.
